#  Movie Recommendation System Using Machine Learning

##  Project Overview
This project implements a **Hybrid Movie Recommendation System** that provides personalized movie suggestions using **Content-Based Filtering** and **Collaborative Filtering** techniques.  
The system analyzes movie metadata and user ratings to recommend relevant movies.

The application is developed using **Python** and deployed using **Streamlit** for an interactive web interface.

---

##  Features
- Content-Based Filtering using **TF-IDF Vectorization**
- Collaborative Filtering using **K-Nearest Neighbors (KNN)**
- Hybrid recommendation combining both approaches
- Interactive **Streamlit** user interface
- Ratings distribution visualization

---

##  Machine Learning Techniques Used
- TF-IDF Vectorization
- Cosine Similarity
- K-Nearest Neighbors (KNN)
- Hybrid Recommendation Strategy
